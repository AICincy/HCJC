# Recipient System Reference - Ohio Project Multi-Recipient Complaint Package

**Last verified:** May 6, 2026
**Verification sources:** codes.ohio.gov; ecfr.gov; uscode.house.gov; hhs.gov/ocr; med.ohio.gov; odh.ohio.gov

## Recipient Matrix

### 1. State Medical Board of Ohio

| Field | Value |
|-------|-------|
| Authority | Ohio Rev. Code 4731 (Medical Practice Act); Ohio Admin. Code 4731 |
| Disciplinary jurisdiction | Ohio Rev. Code 4731.22 (grounds for discipline) |
| Complaint mechanism | Ohio Admin. Code 4731-13-15 |
| Filing address | 30 East Broad Street, 3rd Floor, Columbus, OH 43215 |
| Online filing | med.ohio.gov/Complaints |
| Primary claims | Standard of care violations; provider misconduct; impairment; fraudulent practice |
| Relief available | Investigation; license action (suspension, revocation, restriction); citation; consent agreement |
| Relief NOT available | Monetary damages; record correction; injunctive relief beyond licensure |
| Confidentiality | Complaint and investigation are confidential under Ohio Rev. Code 4731.22(F)(5) until formal action |
| Statute of limitations | Generally none for licensure complaints; practical staleness at 5+ years |
| Attestation | Standard signature; no perjury declaration required |

**Drafting rules.** Frame the complaint as a request for investigation, not as a demand for discipline. State that the complainant requests the Board determine whether the cited conduct violates the Medical Practice Act. Cite specific subsections of Ohio Rev. Code 4731.22 that describe the conduct alleged. Do not cite federal authorities except where federal law incorporates by reference (HIPAA, ADA). Do not request monetary relief.

### 2. HHS Office for Civil Rights, Region V

| Field | Value |
|-------|-------|
| Authority | 45 C.F.R. Part 160 (general administrative requirements); 45 C.F.R. Part 164 (privacy and security rules) |
| Privacy Rule citation | 45 C.F.R. 164.502 (uses and disclosures); 45 C.F.R. 164.524 (right of access) |
| Security Rule citation | 45 C.F.R. 164.302-.318 |
| Breach Notification | 45 C.F.R. 164.400-.414 |
| Region V scope | IL, IN, MI, MN, OH, WI |
| Filing portal | ocrportal.hhs.gov/ocr/complaint |
| Filing address | U.S. Department of Health and Human Services, Office for Civil Rights, 233 N. Michigan Ave, Suite 240, Chicago, IL 60601 |
| Primary claims | Unauthorized use or disclosure; access denial under 45 C.F.R. 164.524; security incident; breach notification failure; record fabrication; failure to amend |
| Relief available | Investigation; corrective action plan; civil monetary penalty against covered entity; resolution agreement |
| Relief NOT available | Monetary recovery to complainant; criminal prosecution (referred separately) |
| Confidentiality | Investigation confidential; resolution may be public |
| Statute of limitations | 180 days from date complainant knew or should have known of the violation, per 45 C.F.R. 160.306(b)(3); extendable for good cause |
| Attestation | Online portal collects under penalty of perjury; written submission requires signed declaration |

**Drafting rules.** Frame each violation as a Privacy Rule, Security Rule, or Breach Notification Rule violation. Cite the exact 45 C.F.R. subsection. Do not allege criminal HIPAA violation under 42 U.S.C. 1320d-6 (criminal HIPAA is referred to DOJ separately and dilutes the OCR civil track). State the date complainant first learned of each violation to establish 180-day timeliness.

### 3. Ohio Department of Health

| Field | Value |
|-------|-------|
| Authority | Ohio Rev. Code 3701; Ohio Admin. Code 3701 |
| Hospital licensure | Ohio Rev. Code 3727 |
| Health Care Facilities Bureau | Ohio Admin. Code 3701-83 et seq. |
| Filing address | Bureau of Survey and Certification, Ohio Department of Health, 246 N. High Street, Columbus, OH 43215 |
| Online complaint | odh.ohio.gov/know-our-programs/health-care-facilities-and-services |
| Primary claims | Institutional policy failures; supervision failures; facility deficiencies; failure to comply with licensure conditions |
| Relief available | Investigation; survey and certification action; deficiency citation; licensure action; corrective action plan |
| Relief NOT available | Individual provider discipline (refer to Medical Board); monetary recovery to complainant |
| Statute of limitations | None statutory; practical staleness at 2+ years |
| Attestation | Standard signature |

**Drafting rules.** Direct allegations at the institution as licensee, not at named individuals. Cite licensure conditions or facility-conduct rules from Ohio Admin. Code 3701-83. Include a chronology demonstrating systemic pattern, not isolated incident.

### 4. Ohio Pre-Suit Notice of Intent

| Field | Value |
|-------|-------|
| Authority | Ohio Rev. Code 2305.113 (medical claim statute of limitations and notice) |
| Tolling provision | Ohio Rev. Code 2305.113(B) (180-day notice tolls one-year SOL) |
| Period | One year from cause of action accrual under 2305.113(A); or four years repose under 2305.113(C) |
| Service requirement | Written notice by certified mail to each prospective defendant prior to expiration of one-year period |
| Recipient | Each individual provider and institution to be named as defendant |
| Required content | Identity of claimant; identity of providers and institutions; statement that civil claim contemplated under 2305.113 |
| Tolling effect | 180 calendar days from date of service of notice |
| Attestation | None for notice itself; civil complaint will require verification |

**Drafting rules.** Notice triggers tolling regardless of substantive merit. Send to every prospective defendant by certified mail with return receipt. Do not include detailed allegations; the notice is procedural. Reserve detailed allegations for the civil complaint.

**ADA and Section 504 considerations.** ADA Title III claims (42 U.S.C. 12181 et seq.) and Section 504 of the Rehabilitation Act (29 U.S.C. 794) may attach to medical claim packages where disability discrimination is alleged. ADA Title III is enforced primarily through DOJ; Section 504 attaches to recipients of federal financial assistance (most hospitals).

### 5. Institution-Specific Internal Grievance

| Field | Value |
|-------|-------|
| Authority | Each institution's published grievance policy (typically Joint Commission RI.01.07.01 mandate) |
| Mechanism | Patient Relations or Patient Advocate office |
| Joint Commission requirement | Institutions must publish grievance procedure; respond in writing within 7 days of receipt |
| CMS Conditions of Participation | 42 C.F.R. 482.13(a)(2) (hospital grievance process) |
| Primary claims | Internal policy escalation; preservation notice; informal resolution attempt |
| Relief available | Internal investigation; service recovery; record correction; staff counseling; institutional response letter |
| Relief NOT available | Monetary damages; provider discipline; regulatory action |
| Confidentiality | Internal; not public |
| Attestation | None required |

**Drafting rules.** Treat the internal grievance as a preservation notice and exhaustion-of-remedies marker. State that the grievance is being filed contemporaneously with external regulatory complaints. Include records-preservation language. Do not waive any external rights by filing internally.

## Filing Order

Recommended sequence when filing all five in coordination:

1. **Internal grievance** - preserves records, exhausts informal remedies
2. **Ohio Department of Health** - initiates institutional review
3. **State Medical Board of Ohio** - initiates individual licensure review
4. **HHS OCR Region V** - initiates federal privacy and access review
5. **Pre-Suit Notice of Intent** - tolls civil SOL; preserves civil track

The internal grievance precedes external filings to capture institutional response before regulatory pressure changes the institutional posture. The pre-suit notice anchors the package because it carries the strictest timing requirement.

## Coordination Rules Across the Package

The five recipient letters share a single factual chronology. Variation across letters is restricted to:
- Jurisdiction statement (recipient-specific authority)
- Legal basis (recipient-specific statutes and regulations)
- Relief requested (recipient-specific remedies)
- Exhibit scope (recipient-specific document subset)

The factual narrative, dated chronology, actor identification, and harm statement remain constant across all five.

## Authority Currency Notes

Citations verified May 6, 2026. The Ohio Revised Code and Ohio Administrative Code are reviewed periodically; recheck via codes.ohio.gov before filing. The C.F.R. is updated annually in print and continuously in eCFR. Federal statutes change by act of Congress; recheck via uscode.house.gov for any cited section before filing. HHS OCR guidance evolves; recheck hhs.gov/hipaa for any cited subregulatory guidance.

The 180-day OCR window under 45 C.F.R. 160.306(b)(3) is the most time-critical element of the package. File OCR within 180 days of discovery or include a written request for good-cause extension.
